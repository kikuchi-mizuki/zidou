# チョコレート卸向け納品計画WEBアプリ

デパートからの**海外高級チョコレート**発注に対して、在庫・賞味期限・納品ルールを考慮し、最適な納品計画を自動生成するシステムのドキュメントサイトです。

## 🍫 システム概要

このシステムは、チョコレート卸売業者がデパートからの発注に対して効率的な納品計画を作成するためのツールです。

## 🚀 クイックスタート

### デモアプリケーションを試す

1. リポジトリをクローン
```bash
git clone [repository-url]
cd chocolate-delivery-app
```

2. 依存パッケージをインストール
```bash
npm install
```

3. 開発サーバーを起動
```bash
npm run dev
```

4. ブラウザでアクセス
```
http://localhost:3000
```

## 💡 主な機能

### 1. ダッシュボード
発注件数、在庫アラート、期限が近い在庫の一覧表示

### 2. 商品管理
取り扱い商品の一覧表示

### 3. 在庫管理
在庫の追加・編集・削除、ケース数の直接編集

### 4. 発注管理
新規発注の登録、発注の削除、ステータス管理

### 5. ルール管理
ルールの追加、有効/無効切り替え

### 6. 自動計算
実際の計算ロジック実装、リアルタイム計算ログ表示

### 7. 納品計画
日付別グループ表示、計画の承認機能
