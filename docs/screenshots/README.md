## 撮影方法

### Macの場合
1. `Command + Shift + 4` でスクリーンショットモード
2. 撮影したい範囲を選択
3. デスクトップに保存される

### Windowsの場合
1. `Windows + Shift + S` でスクリーンショットモード
2. 撮影したい範囲を選択
3. クリップボードに保存される

### ブラウザの開発者ツール（推奨）
1. F12で開発者ツールを開く
2. `Command/Ctrl + Shift + P` でコマンドパレット
3. "screenshot" と入力
4. "Capture full size screenshot" を選択
5. ダウンロードフォルダに保存される

## 撮影後の配置

撮影したスクリーンショットを以下のフォルダに配置：

docs/screenshots/
